# Phore bootstrap base theme

- Includes 
    - [JQuery 3.3.1]()
    - [FontAwesome 5.1]() 
    - [Bootstrap 4.1]()
    - [Popper]()
    
Libraries are included as assets

## Installation

```
composer require phore/theme-bootstrap
```

## Example

Example / testing is included. Run `kickstart.sh` to go live and open
`http://localhost` in your browser.
